### Mục tiêu Tuần 7:

* Nắm vững các dịch vụ Cơ sở dữ liệu AWS - Database Services:
  * Amazon RDS (Relational Database Service) - quản lý database quan hệ
  * Amazon Aurora - MySQL và PostgreSQL tương thích với hiệu suất cao
  * Amazon Redshift - data warehouse cho analytics quy mô lớn
  * Amazon ElastiCache (Redis, Memcached) - in-memory caching
  * Phân biệt Online Transaction Processing (OLTP) vs Online Analytical Processing (OLAP)
* Phát triển kỹ năng AI thực tế - ứng dụng thực tế:
  * Nghiên cứu các loại Chatbot và use cases
  * Thiết kế conversation flow và intent detection
  * Tìm hiểu các nền tảng AI: Amazon Lex, Amazon Comprehend
  * Xây dựng prototype AI Chatbot bằng Python
  * Tích hợp Chatbot vào kiến trúc dự án
* Hoàn thiện Sơ đồ Kiến trúc Tổng quan:
  * Điều chỉnh sơ đồ dựa trên feedback của mentor
  * Bổ sung các dịch vụ database và AI vào kiến trúc
  * Chốt phiên bản cuối cùng của system design
* Bắt đầu triển khai Dự án Cuối khóa - Implementation phase:
  * Thiết kế Frontend (UI/UX) với React hoặc Vue.js
  * Thiết kế Backend API architecture với RESTful hoặc GraphQL
  * Thiết lập môi trường development và Git workflow
  * Bắt đầu coding các module cơ bản
