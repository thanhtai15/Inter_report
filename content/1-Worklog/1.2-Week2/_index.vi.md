### Kết quả đạt được tuần 2:

* Hiểu rõ về các thành phần cốt lõi của VPC:
  * CIDR Block và cách phân chia địa chỉ IP
  * Subnet (Public vs Private)
  * Route Table và Internet Gateway
  * VPN, DirectConnect, Load Balancer

* Thực hành thành công:
  * Tạo và cấu hình VPC với các Subnets
  * Thiết lập Route Table và Internet Gateway
  * Hiểu sự khác biệt giữa Public và Private Subnet

* Nắm vững Security Group và Network ACLs:
  * Security Group: Stateful, hoạt động ở tầng instance
  * Network ACLs: Stateless, hoạt động ở tầng subnet
  * Cách cấu hình rules cho inbound và outbound traffic

* Làm quen với VPC Resource Map để trực quan hóa kiến trúc mạng

* Gặp gỡ và làm quen với các thành viên trong nhóm dự án cuối kỳ
