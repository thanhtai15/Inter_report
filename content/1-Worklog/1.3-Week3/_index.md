### Week 3 Objectives:

* Deploy and manage EC2 Instances in complex VPC environment:
  * Create EC2 Instances in different Subnets
  * Configure Transit Gateway to connect multiple VPCs
  * Set up VPC Endpoints for private connections
* Master advanced EC2 concepts:
  * Amazon Machine Images (AMI) and Backup management
  * Distinguish between EBS (Elastic Block Store) and Instance Store
  * EC2 Auto Scaling for automatic system expansion
  * User Data and Meta Data in EC2
* Get familiar with AWS Storage and Content Delivery:
  * Create and manage S3 Bucket
  * Host static website on S3
  * Use CloudFront CDN to speed up access
* Begin Machine Learning and NLP journey:
  * Learn about Supervised Machine Learning
  * Sentiment Analysis and Natural Language Processing
  * Build simple Logistic Regression models
