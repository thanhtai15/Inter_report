### Mục tiêu tuần 3:

* Triển khai và quản lý EC2 Instances trong môi trường VPC phức tạp:
  * Tạo EC2 Instances trong các Subnets khác nhau
  * Cấu hình Transit Gateway để kết nối nhiều VPCs
  * Thiết lập VPC Endpoints cho kết nối riêng tư
* Nắm vững các khái niệm EC2 nâng cao:
  * Amazon Machine Images (AMI) và quản lý Backup
  * Phân biệt EBS (Elastic Block Store) và Instance Store
  * EC2 Auto Scaling để tự động mở rộng hệ thống
  * User Data và Meta Data trong EC2
* Làm quen với AWS Storage và Content Delivery:
  * Tạo và quản lý S3 Bucket
  * Host website tĩnh trên S3
  * Sử dụng CloudFront CDN để tăng tốc truy cập
* Bắt đầu hành trình Machine Learning và NLP:
  * Học về Supervised Machine Learning
  * Sentiment Analysis và xử lý ngôn ngữ tự nhiên
  * Xây dựng mô hình Logistic Regression đơn giản
