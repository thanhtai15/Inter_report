#### Create Public Subnet

1. Open the [Amazon VPC console](https://ap-southeast-1.console.aws.amazon.com/vpcconsole/home?region=ap-southeast-1#Home:)
2. Choose **Subnets**  -> click **Create subnet**
![endpoint](/images/Subnets/s1.png)
1. In Create subnet console:
- Choose VPC created
- Fill subnet name
- Choose AZ
- Spacific IPv4 subnet
![endpoint](/images/Subnets/s2.png)
1. Then click **Create subnet**
![endpoint](/images/Subnets/s3.png)

#### Create Private Subnet
1. Do again step 1 -> 4
2. Click this Subnet -> Choose Route table
![endpoint](/images/Subnets/s4.png)
3. Choose Route table ID is private
4. Click **Save**
![endpoint](/images/Subnets/s5.png)
