#### Overview
- This project used three data storage services which are:
  + AWS RDS(Relational Database Service) is a AWS service to config database and is located in Private Subnet
  + AWS S3 (Simple Storage Service) is a AWS service to storage picture and video
  + AWS ECR (Elastic Container Registry) is a AWS service to storage docker image
