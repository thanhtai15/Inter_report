Worklog Tuần 5

Mục tiêu Tuần 5:

  • Nắm vững các thuật toán NLP nâng cao và Machine Learning:
    • Biến đổi Vector (Vector Transformation) để dịch thuật từ
    • K-nearest neighbors (KNN) cho phân loại văn bản
    • Hash tables và hash functions để tìm kiếm nhanh
    • Thực hành lab Word Translation với word embeddings
  • Đi sâu vào hệ sinh thái AWS Storage - lưu trữ dữ liệu toàn diện:
    • Các tính năng nâng cao của S3: Access Points, Storage Classes, Versioning
    • S3 Static Website hosting và CORS configuration
    • Object Key naming và Performance optimization
    • Amazon Glacier cho lưu trữ dài hạn giá rẻ
    • AWS Snow Family (Snowcone, Snowball, Snowmobile) cho data migration
    • AWS Backup cho chiến lược sao lưu tập trung
  • Triển khai tuân thủ Bảo mật và Tự động hóa:
    • AWS Security Hub để đánh giá compliance
    • IAM Roles và Policies cho EC2 và Lambda
    • AWS Lambda functions để tự động hóa tác vụ
    • Resource tagging strategy cho quản lý và cost allocation
  • Hoàn thiện kế hoạch thực hiện chi tiết cho dự án cuối kỳ:
    • Xác định kiến trúc hệ thống
    • Phân chia công việc và timeline
    • Chọn công nghệ và dịch vụ AWS sử dụng

Thành tựu Tuần 5:

  • Nắm vững số học Không gian Vector trong NLP:

    • Triển khai KNN và Bảng băm.
    • Lập trình thành công mô hình Dịch thuật Từ sử dụng biến đổi vector.

  • Cấu hình các Kiến trúc Lưu trữ AWS Nâng cao:

    • Triển khai các Website tĩnh trên S3 với cấu hình CORS.
    • Quản lý các Lớp lưu trữ S3 (Storage Classes) và Kiểm soát truy cập.
    • Hiểu về di chuyển dữ liệu ngoại tuyến sử dụng AWS Snow Family.

  • Triển khai Bảo mật Đám mây và Tự động hóa:

    • Kích hoạt và đánh giá điểm tuân thủ sử dụng AWS Security Hub.
    • Tạo các quy trình tự động hóa Serverless sử dụng AWS Lambda và IAM Roles.
    • Quản lý tài nguyên EC2 thông qua các chiến lược gắn thẻ (tagging).

  • Tiến độ Dự án:

    • Hoàn thành chuỗi bài thực hành mở rộng (Module 05).
    • Hoàn thiện và thống nhất lộ trình chi tiết cho dự án cuối khóa.
